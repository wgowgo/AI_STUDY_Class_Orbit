# 공공 교육 데이터 기반 AI 시스템 — 심층 개선 제안서

> 작성일: 2026년 4월 1일
> 대상: 전 연령/평생교육 | 학생, 학부모 및 교사, 교육자 타겟
> 현재 활용 데이터: 학업 성취도/성적 데이터, 학교/기관 정보

---

## 1. 추가 핵심 기능 제안

### 1-1. 학습 경로 자동 설계 엔진 (Learning Pathway Engine)

현재 성취도 데이터를 단순 조회하는 수준에서 벗어나, **개인별 학습 경로를 자동으로 설계**하는 기능이 필요합니다. 학생의 현재 성취 수준, 목표, 학습 속도를 분석하여 다음에 무엇을 공부해야 하는지를 구체적으로 제시하는 것입니다. 예를 들어 수학에서 분수 개념이 약한 학생에게는 분수 관련 선행 개념부터 단계적으로 안내하고, 충분히 숙달된 영역은 건너뛰도록 합니다.

### 1-2. 학습 격차 조기 경보 시스템 (Early Warning System)

공공 데이터의 성취도 패턴을 분석하여 **학습 부진 위험이 있는 학생을 조기에 식별**하는 기능입니다. 출석률, 성적 추이, 과목별 편차 등 여러 지표를 종합적으로 모니터링하여 교사와 학부모에게 사전 경고를 보냅니다. OECD 2026 디지털 교육 전망에서도 AI의 교육적 활용에서 가장 가치 있는 방향 중 하나로 예방적 개입을 꼽고 있습니다.

### 1-3. 교육 자원 매칭 플랫폼

학교/기관 정보와 지역별 교육 자원 데이터를 결합하여, **학생에게 가장 적합한 교육 기회를 추천**하는 기능입니다. 방과후 프로그램, 장학금, 온라인 강좌, 지역 도서관 프로그램 등을 학생의 관심사와 수준에 맞춰 연결합니다.

### 1-4. 다국어 지원 및 접근성 강화

전 연령 대상 플랫폼인 만큼, 다문화 가정 학생을 위한 다국어 인터페이스와 장애 학생을 위한 접근성 기능(음성 안내, 화면 확대, 간편 UI 등)을 핵심 기능으로 포함해야 합니다.

## 2. 공공 교육 데이터 활용 확장 방안

### 2-1. 추가 확보 가능한 공공 데이터 유형

현재 활용 중인 학업 성취도와 학교 정보 외에, 다음 데이터를 통합하면 시스템의 가치가 크게 높아집니다.

| 데이터 유형 | 출처 예시 | 활용 방향 |
|---|---|---|
| **교육과정 데이터** | 교육부 국가교육과정 정보센터 | 교과 연계 학습 경로 설계 |
| **지역별 교육 통계** | 한국교육개발원(KEDI) 교육통계 | 지역 간 교육 격차 분석 |
| **대학 입시 결과** | 대학알리미, 대입정보포털 | 진로, 진학 예측 및 추천 |
| **직업, 고용 데이터** | 한국고용정보원, 워크넷 | 진로 탐색과 평생교육 연계 |
| **학교 평가 인증 정보** | 학교정보공시(학교알리미) | 학교 선택 의사결정 지원 |
| **EBS/공공 학습 콘텐츠** | EBS, K-MOOC, KOCW | 무료 학습 자원 추천 |
| **특수교육 통계** | 국립특수교육원 | 특수교육 대상 맞춤 지원 |

### 2-2. 데이터 통합 전략

단순히 데이터를 나열하는 것이 아니라, **데이터 간 연결 관계를 만드는 것**이 핵심입니다.

**종단 연결**: 초등→중등→고등→대학→취업에 이르는 학습 이력의 흐름을 추적하여, 어떤 학습 경로가 어떤 결과로 이어지는가를 분석할 수 있는 구조를 만듭니다. 이는 진로 상담과 교육 정책 수립에 매우 강력한 도구가 됩니다.

**횡단 연결**: 같은 시점에서 지역, 학교 유형, 사회경제적 배경 등 다양한 변수를 교차 분석하여 교육 불평등의 패턴을 발견하고, 어디에 자원을 집중해야 하는지 판단할 수 있습니다.

**실시간 연동**: 서울시교육청의 AI 교수학습 플랫폼처럼, 수업 중 생성되는 학습 데이터를 실시간으로 수집, 분석하는 파이프라인을 구축하면 즉각적인 피드백이 가능해집니다.

## 3. 핵심 AI 기능 제안

### 3-1. 예측 분석 (Predictive Analytics)

**학업 성취 예측**: 과거 성적 추이, 출석률, 과제 참여도 등을 기반으로 향후 학업 성취도를 예측합니다. 단순 점수 예측이 아니라, 어떤 과목에서 어려움을 겪을 가능성이 높은지, 어떤 개입이 가장 효과적일지까지 제안합니다.

**중도 탈락 예측**: 특히 고등교육과 평생교육 영역에서, 학습 이탈 위험이 있는 학습자를 조기에 식별하여 맞춤형 지원을 제공합니다.

**진로 적합도 예측**: 학업 성취 패턴, 관심 분야, 지역 노동시장 데이터를 결합하여 학생에게 적합한 진로를 추천합니다.

### 3-2. 적응형 추천 시스템 (Adaptive Recommendation)

넷플릭스가 시청 이력으로 콘텐츠를 추천하듯, **학습 이력 기반으로 최적의 학습 자원을 추천**합니다. 여기서 차별점은 공공 데이터를 활용하므로 무료 또는 공공 학습 자원 중심으로 추천할 수 있다는 것입니다. EBS 강의, K-MOOC 과정, 지역 도서관 프로그램 등 비용 부담 없이 접근 가능한 자원을 우선 추천합니다.

### 3-3. AI 튜터링 기능

공공 교육과정 데이터와 연계된 **AI 튜터**를 제공합니다. 일반적인 챗봇과 다른 점은, 국가 교육과정에 맞춰 정확한 학년, 단원별 설명을 제공하고, 학생의 현재 수준에 맞춰 난이도를 조절한다는 것입니다. 교과서 진도와 동기화되어 오늘 배운 내용을 복습하거나 내일 배울 내용을 예습하는 식의 활용이 가능합니다.

### 3-4. 교육 데이터 시각화 및 인사이트 대시보드

교사용으로는 학급, 학교 단위의 학습 현황을 한눈에 파악할 수 있는 대시보드를, 학부모용으로는 자녀의 학습 진행 상황과 또래 대비 위치를 직관적으로 보여주는 리포트를 제공합니다. 단, 또래 비교는 순위 경쟁이 아닌 강점 영역과 성장 기회 영역 중심으로 설계하여 건강한 학습 동기를 유도합니다.

### 3-5. 자연어 데이터 질의 (Natural Language Query)

교사나 교육 연구자가 우리 학교 3학년 수학 성취도가 지난 3년간 어떻게 변했나요?와 같은 자연어 질문으로 데이터를 조회하고 분석 결과를 받을 수 있는 기능입니다. 복잡한 데이터 분석 도구를 배우지 않아도 누구나 교육 데이터의 인사이트를 활용할 수 있게 합니다.

## 4. 사용자 그룹별 맞춤 전략

### 4-1. 학생

| 학생 유형 | 핵심 니즈 | 제공 기능 |
|---|---|---|
| 초등학생 | 흥미 유발, 기초 학력 | 게이미피케이션, AI 튜터 (쉬운 인터페이스) |
| 중학생 | 자기주도 학습 습관 | 학습 계획 자동 생성, 취약점 분석 |
| 고등학생 | 입시 준비, 진로 탐색 | 성적 예측, 대학/학과 추천, 수능 대비 |
| 대학생/성인 | 취업, 역량 개발 | 직무 역량 매칭, 자격증 추천 |
| 시니어 | 디지털 리터러시, 취미 | 쉬운 UI, 음성 지원, 사회 참여 연계 |

### 4-2. 학부모

학부모에게는 **내 아이에게 맞는 교육 의사결정을 돕는 도구**가 핵심입니다. 학교 선택 시 학교알리미 데이터를 기반으로 학교별 특성을 비교하거나, 자녀의 학습 현황을 주간 리포트로 받아볼 수 있습니다. 사교육에 의존하지 않고도 공공 교육 자원만으로 충분한 학습 지원이 가능하다는 것을 보여주는 것이 가치 제안의 핵심입니다.

### 4-3. 교사/교육자

교사에게는 **행정 부담 경감**과 **수업 질 향상**이라는 두 가지 가치를 동시에 제공해야 합니다. 성적 데이터 기반의 자동 학급 분석, 학생별 맞춤 피드백 초안 생성, 교육과정 재구성 지원 등이 포함됩니다. 2025년 기준으로 이미 85%의 교사가 AI를 수업에 활용하고 있으므로, 기존 도구(하이러닝, 서울시교육청 플랫폼 등)와의 연동성이 중요합니다.

### 4-4. 추가 고려 사용자: 교육 정책 결정자

직접적인 타겟은 아니더라도, 교육청이나 교육 연구 기관이 활용할 수 있는 **정책 시뮬레이션 기능**을 제공하면 B2G(정부 대상 사업) 모델로 확장 가능합니다.

## 5. 서비스 모델 및 플랫폼 구조

```
┌─────────────────────────────────────────────┐
│              사용자 인터페이스 계층             │

├─────────────────────────────────────────────┤
│              AI 서비스 계층                    │

│            데이터 통합 계층                    │

└─────────────────────────────────────────────┘
```

### 5-2. 수익 모델 제안

**프리미엄(Freemium) 모델**이 가장 적합합니다.

**무료 계층**: 공공 데이터 기반 기본 분석, 학교 정보 조회, 기본 AI 튜터, 학습 경로 추천 — 공공 데이터를 활용하므로 기본 서비스는 무료로 제공하여 사회적 가치를 창출합니다.

**유료 계층 (월 구독)**: 심화 예측 분석, 개인 맞춤 상세 리포트, 1:1 AI 튜터링 시간 확대, 진로 상담 AI 고도화 기능을 제공합니다.

**B2B/B2G 계층**: 학교, 교육청 대상 기관 라이선스로, 학급 관리 대시보드, 정책 시뮬레이션, 대량 데이터 분석 기능을 포함합니다.

### 5-3. 오픈 API 생태계

핵심 데이터 분석 기능을 API로 공개하여, 외부 에듀테크 기업이나 개발자가 자사 서비스에 통합할 수 있도록 합니다. 이는 플랫폼의 영향력을 확대하고 교육 데이터 활용 생태계를 키우는 전략입니다.

## 6. 차별화 아이디어

### 6-1. 교육 형평성 지수 도입

가장 강력한 차별화 포인트로, 공공 데이터를 활용하여 **지역, 학교, 개인 수준의 교육 형평성 지수**를 산출하고 시각화하는 기능을 제안합니다. 기존 교육 플랫폼이 더 높은 점수에 초점을 맞추는 반면, 이 시스템은 모든 학생이 공정한 교육 기회를 받고 있는가라는 질문에 답합니다. 이는 사회적 가치와 데이터의 독창적 활용을 동시에 달성합니다.

### 6-2. 크라우드소싱 학습 인사이트

학생과 교사가 이 단원에서 이 방법이 효과적이었다는 학습 팁을 공유하고, AI가 이를 검증, 정제하여 다른 사용자에게 추천하는 구조입니다. 공공 데이터의 객관적 분석과 실제 학습 현장의 경험적 지식이 결합되는 것입니다.

### 6-3. 학부모 교육 내비게이션

네비게이션 앱처럼, 학부모가 우리 아이의 현재 위치와 목표 지점을 설정하면, AI가 최적의 교육 경로를 안내합니다. 공공 데이터 기반이므로 사교육 추천이 아닌 공교육 자원 중심의 경로를 제시한다는 점이 기존 서비스와 근본적으로 다릅니다.

### 6-4. 교육 데이터 스토리텔링

딱딱한 통계가 아닌, **우리 동네 교육 이야기** 형태로 지역별 교육 현황을 이야기로 풀어내는 기능입니다. ○○구의 초등학교 독서 활동이 3년간 40% 증가했고, 이는 국어 성취도 향상과 함께 나타났습니다와 같은 내러티브를 AI가 자동 생성합니다.

### 6-5. 디지털 격차 해소 모드

인터넷 환경이 열악한 지역을 위한 **오프라인 우선(Offline-first) 모드**를 제공합니다. 핵심 AI 기능의 경량 버전을 로컬에서 실행할 수 있도록 하여, 디지털 격차가 교육 격차로 이어지지 않도록 합니다.

## 7. 구현 우선순위 제안

단계적 접근을 권장합니다.

**1단계 (MVP, 3-6개월)**: 공공 데이터 통합 파이프라인 구축, 학교 정보 검색 및 비교, 기본 성취도 분석 대시보드, 학습 자원 추천 기본 기능

**2단계 (핵심 AI, 6-12개월)**: 학습 경로 자동 설계 엔진, 학습 격차 조기 경보 시스템, AI 튜터 기본 버전, 교사용 학급 분석 대시보드

**3단계 (확장, 12-18개월)**: 진로 예측 및 추천 고도화, 교육 형평성 지수, 자연어 데이터 질의, 오픈 API 공개

**4단계 (생태계, 18개월 이후)**: B2G 서비스 확장, 크라우드소싱 학습 인사이트, 오프라인 모드, 평생교육 연계 강화
